# Incident Report

## Incident ID

SOC-2026-002
## Title

Process Creation Investigation Using Sysmon

## Analyst

Salat Abdullahi Issak

## Date

20 July 2026
## Severity
Informational (Lab Simulation)

## Summary

A controlled laboratory exercise was conducted to analyze Sysmon Event ID 1 records generated by common Windows applications.

The objective was to understand how Sysmon records process execution and how these logs can support endpoint investigations.

## Applications Investigated

- Notepad
- Calculator
- Command Prompt
- PowerShell

## Evidence

Primary Event

Sysmon Event ID 1

Example Process

notepad.exe

User

FINANCE-PC-07\SOC1

Parent Process

explorer.exe

Integrity Level

Medium

Command Line

"C:\Windows\System32\notepad.exe"



## Analysis

The investigation confirmed that Notepad was launched interactively by the SOC1 user through Windows Explorer.

The executable path matched the legitimate Microsoft binary.

No suspicious command-line arguments, parent processes or privilege escalation indicators were identified.

The SHA256 hash was successfully recorded by Sysmon, demonstrating how executable integrity can be verified during investigations.

## Conclusion

The observed activity was determined to be legitimate user activity within the SOC home lab.

No indicators of malicious execution were identified.
